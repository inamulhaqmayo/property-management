## 🚀 GET STARTED IN 10 MINUTES

### Step 1: Database Setup (3 min)
1. Go to https://neon.tech
2. Sign up (free tier available)
3. Create a new PostgreSQL project
4. Copy your connection string

### Step 2: Configure Environment (2 min)
```bash
cd /Users/inamulhaqmayo/Desktop/inam-portfolio
cp .env.example .env.local
# Edit .env.local and add your DATABASE_URL
```

### Step 3: Initialize Database (3 min)
```bash
npm run seed
```

This creates:
- Database tables
- Admin user (admin@inamulhaq.com / demo123456)
- Sample portfolio items
- Sample testimonials
- Sample services
- Sample blog posts
- Site settings

### Step 4: Start Development (2 min)
```bash
npm run dev
```

Visit:
- **Homepage:** http://localhost:3000
- **Admin:** http://localhost:3000/admin
- **Login:** admin@inamulhaq.com / demo123456
