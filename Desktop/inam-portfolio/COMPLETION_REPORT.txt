================================================================================
                         API ENDPOINTS
================================================================================

AUTHENTICATION:
  POST   /api/auth/login       - Login with email/password
  POST   /api/auth/logout      - Logout and clear session
  GET    /api/auth/verify      - Verify current session

PORTFOLIO:
  GET    /api/portfolio        - Get all items (with ?featured=true)
  POST   /api/portfolio        - Create new item
  GET    /api/portfolio/[id]   - Get single item by ID or slug
  PUT    /api/portfolio/[id]   - Update item
  DELETE /api/portfolio/[id]   - Delete item

TESTIMONIALS:
  GET    /api/testimonials     - Get all testimonials
  POST   /api/testimonials     - Create new testimonial
  GET    /api/testimonials/[id] - Get single testimonial
  PUT    /api/testimonials/[id] - Update testimonial
  DELETE /api/testimonials/[id] - Delete testimonial

SERVICES:
  GET    /api/services         - Get all services
  POST   /api/services         - Create service
  GET    /api/services/[id]    - Get single service
  PUT    /api/services/[id]    - Update service
  DELETE /api/services/[id]    - Delete service

BLOG:
  GET    /api/blog             - Get published posts (or all with ?includeUnpublished=true)
  POST   /api/blog             - Create post
  GET    /api/blog/[id]        - Get single post
  PUT    /api/blog/[id]        - Update post
  DELETE /api/blog/[id]        - Delete post

CONTACT:
  POST   /api/contact          - Submit contact form (sends email + saves to DB)
  GET    /api/contact          - Get all messages (admin only)

SETTINGS:
  GET    /api/settings         - Get all settings (public)
  PUT    /api/settings         - Update settings (admin only)
